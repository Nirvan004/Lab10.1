This is a React Counter that utilizes useState and useEffect to perform the desired functions.

It defaultly increments or decrements by 1.
The value of movement can be changed with the "step" input 
it holds previous counts an displays them in the "previous counts" section
the counter can be reset back to 0 also reseting the previous counts
It automatically saes the count to local stoarage when a button (up or down) is clicked 
Also uses event listeners to let users utilize the arrow up and arrow down key to count
